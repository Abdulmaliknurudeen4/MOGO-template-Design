# How to Install/Use
In Order to Run this Jekyll Project, Follow these steps:
* Install Ruby
* Then Run in Your Terminal `gem install jekyll`
* Change your Working Directory to the Project Path
* Run `jekyll serve`
* Opps , It's not highly Responsive, an Old Project
* Enjoy
